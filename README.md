# kTab Manager

kTab Manager is a keyboard-first Firefox extension for managing tabs. It shows all your open tabs across all windows in one clean, fast view.

## Keyboard flow

The core experience is designed to never leave the keyboard:

- **Option+T** (Mac) or **Alt+Shift+T** (Windows/Linux) — open kTab Manager
- **Type** to instantly filter by domain or page title
- **↑ / ↓** to navigate results
- **Enter** to switch to the selected tab
- **Option+X** (Mac) or **Alt+X** (Windows/Linux) — close the highlighted tab

## Other features

- **Domain / Window toggle** — group tabs by domain (all YouTube tabs together, all GitHub tabs together, etc.) or by browser window
- **Mute / unmute** tabs individually, or mute all playing tabs at once with a single click
- **Close duplicates** — closes all duplicate tabs in one click, keeping one per URL

## Privacy

kTab Manager only requests the `tabs` permission. It makes no external connections and collects no data whatsoever.

## Why another tab manager?

Because I couldn't find what I wanted. As surprising as it seems, I was unable to find a simple Firefox tab manager that let me view all my tabs at once and search and navigate them with just the keyboard.

Every extension I tried was either missing something, too complex, or buried the simple view I needed.

When you have dozens of Claude tabs open and need to find that one conversation from yesterday, you'll love to just hit Option+T, type "clau", and arrow down to it. Done.

Hope you enjoy kTab Manager as much as I do!
